# Dungeon_Diver
Dungeon Diver is a simple Terminal Based Dungeon Game that involves a player collecting loot and going through a dungeon in order to escape. Along the players path other will appear several monsters attempting to kill and prevent the players path out of the dunegon. The hope is to further expand the project into a GUI version sort of like Binding of Issac like with more monsters and bosses.
